## 3. Project and scope
  - Function to generate four basic Waveforms:
    - Sine Wave
    - Square Wave
    - Sawtooth Wave
    - Triangle Wave
  - Function to save waveform to disk
  - Function to read a section of an audio file
  - Function to play the waveform directly
  - Basic Filters:
    - Overdriven filter
    - Echo filter
    - Flange filter
    - Reverb filter
  - Spectroscope
  - Low frequency oscillator
  - MP3 Compression
